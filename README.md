# Unit Convention:
All constants are as specified, unless stated otherwise:
 - Distance: Meters
 - Angle: Degrees
 - Time: Seconds
 - Speed: Meters/second
 - Acceleration: Meters/second^2
 - Angular Speed: radians/second

## Variable Naming Convention:
 - k Final Constants
 - m Private Instance Variables
 - b Xbox Controller Buttons
 - e Enums
